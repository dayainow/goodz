# 승인 로그

Goodz Process OS의 Phase Gate와 주요 스프린트 산출물 승인 기록입니다.  
기계 판독 상태는 `docs/00-process/status.json`의 `approvals` 필드를 SSOT로 사용합니다.

## 승인 내역

| ID | 대상 | 승인자 | 일자 | 상태 | 요약 |
|----|------|--------|------|------|------|
| AP-001 | S4 Process OS v0.3 | 사용자 | 2026-07-10 | approved | 기획 입력함, 산출물 레지스트리, 대시보드 기획·산출물 메뉴 반영 승인 |

## 운영 규칙

1. Phase Gate 또는 스프린트 산출물 승인 시 이 문서와 `status.json`을 함께 갱신합니다.
2. 승인 후 변경 요청이 생기면 새 승인 ID를 만들고 `changes_requested` 또는 `requested` 상태를 남깁니다.
3. 대시보드 `승인` 메뉴에서 최신 승인 상태를 확인합니다.
