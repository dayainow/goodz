# Template Onboarding v0.15 변경

## 요청

S16 이후 작업 단계를 파악하고, Goodz의 v1.0 목표인 fork·판매 가능한 템플릿화를 계속 진행합니다.

## 변경

- Template Onboarding 런북 추가
- template.config.json에 필수 산출물·스크립트·설정 계약 정의
- check:template을 verify와 CI에 연결
- ga-analytics-harness를 로컬 상대경로 대신 GitHub commit pin으로 전환
- ROADMAP, PROJECT, 승인, 결정, 산출물, trace SSOT 갱신

## 결정

v0.15는 v1.0 완료로 표시하지 않고 **Template Onboarding Baseline**으로 관리합니다. clean-clone CI와 실제 rebrand 리허설 증거가 있어야 v1.0으로 승격합니다.
