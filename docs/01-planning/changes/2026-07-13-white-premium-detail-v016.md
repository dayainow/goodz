# White Premium Detail Tuning v0.16 변경

## 요청

화이트 프리미엄 완성도를 높이기 위해 색 온도, 타이포 계층, 카드 깊이, 사이드바 단서, 메트릭 문구를 세부 조정합니다.

## 변경

- Quick Jump 비활성 대비와 메뉴 hover 강화
- 접힘 그룹에 회전형 chevron 단서 추가
- CTA를 primary, signal, health 위계로 분리
- Completion은 완료 badge, Operations는 대기열 없음 문장으로 변경
- P2 현재 운영 ring과 badge 추가
- Sprint, Version, Updated, Section 메타데이터 위계 분리
- Noto Sans KR 우선 스택, heading/body line-height, 6px scrollbar 반영
- progress bar를 near-black으로 정리

## 결정

모든 카드를 같은 색과 크기로 정렬하는 대신 정보 역할에 따라 표면, 그림자, 타이포 크기를 다르게 사용합니다. 상태 색은 완료·주의·위험 신호에만 제한합니다.
