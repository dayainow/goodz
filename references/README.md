# Goodz References

Reference는 Goodz Core가 특정 산업에 종속되지 않음을 증명하는 실행 가능한 예제입니다.

| Reference | 도메인 | 상태 | 목적 |
|---|---|---|---|
| Goodz Commerce Reference | commerce | verified | B2C/운영 UI와 E2E 타입 패턴 |
| [Goodz Internal Service Reference](./internal-service/README.md) | internal-platform | verified | 비커머스 이식성과 독립 타입/API 패턴 |

새 Reference는 자체 타입·앱·P0–P4 문서를 가지며 `packages/process`를 수정하지 않아야 합니다.
